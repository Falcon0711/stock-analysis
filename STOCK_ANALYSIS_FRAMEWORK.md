# 股票分析项目框架与流程

## 项目概述
这是一个专业的股票分析系统，旨在提供全面的股票数据分析、技术指标计算和投资建议。

## 目录结构

```
stock_analysis_project_new/
├── models/           # 核心模型与分析算法
│   ├── core/         # 核心分析模型
│   └── ai/           # AI驱动的分析模型
├── sources/          # 数据源接口
├── utils/            # 工具函数与辅助功能
├── config/           # 配置文件
├── agents/           # 智能体与技能定义
│   └── skills/       # 各类分析技能
├── scripts/          # 执行脚本
├── docs/             # 文档
├── reports/          # 生成的分析报告
├── env/              # Python虚拟环境
└── tests/            # 测试用例
```

## 核心流程

### 1. 数据获取流程
- 通过 `sources/` 中的数据接口获取股票数据
- 支持多种数据源（腾讯、雅虎财经等）
- 数据预处理与清洗

### 2. 分析流程
- 基本面分析（财务数据、公司信息）
- 技术面分析（各类技术指标计算）
- AI辅助分析（使用大型语言模型进行综合判断）

### 3. 报告生成流程
- 结合多维度分析结果
- 生成标准化分析报告
- 提供投资建议

## 关键模块

### 数据源模块 (`sources/`)
- `tencent_data_source.py`: 腾讯财经数据接口
- 其他数据源适配器

### 分析模型 (`models/`)
- `core/` - 核心分析模型
  - `technical_indicators.py` - 技术指标计算（KDJ, MACD, RSI, 均线等）
  - `analyzer.py` - AI分析器
  - `pipeline.py` - 分析流水线
  - `market_review.py` - 大盘复盘分析
- `ai/` - AI驱动的分析模型

### 技能系统 (`agents/skills/`)
- 股票分析技能
- 项目检查技能
- 自动化分析技能

### 工具函数 (`utils/`)
- 数据处理工具
- 报告生成工具
- 通知工具

### 执行脚本 (`scripts/`)
- `main.py`: 主程序入口，完整的分析流程
- `examples/`: 示例脚本
  - `demo_ai_analysis.py`: AI分析框架演示
  - `run_stock_analysis.py`: 简化的股票分析脚本

## 使用指南

### 启动分析
1. 配置数据源和API密钥
2. 运行主分析脚本：`python scripts/main.py`
3. 查看生成的报告在 `reports/` 目录

### 自定义分析
- 在 `agents/skills/` 中添加自定义分析技能
- 修改配置文件以适应特定需求

## 扩展性
- 模块化设计，易于扩展新的数据源
- 插件式技能系统，可灵活添加分析能力
- 支持多种AI模型集成
