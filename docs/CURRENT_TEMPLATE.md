# 当前技术分析报告模板

这是当前实际使用的模板格式，反映了最新的BBI指标替代RSI的要求。

## 标准模板格式

```
=================================================================
              [股票名称]([股票代码]) 技术分析报告
=================================================================
📈 基本信息: [股票名称] | [股票代码]
💰 当前价格: [当前价格]元 | 涨跌: [涨跌额] | 涨幅: [涨跌幅]%

📊 技术指标概览:
  KDJ: K=[K值], D=[D值], J=[J值] | 信号: [金叉/死叉符号]
  MACD: [MACD值], [信号线值], [柱状图值] | 信号: [多头/空头符号]
  BBI: [BBI值] | 位置: [上方/下方]
  MA5/10/20/60: [MA5值]/[MA10值]/[MA20值]/[MA60值]
  知行指标: 趋势线=[趋势线值] | 位置: [上方/下方], 多空线=[多空线值] | 位置: [上方/下方]

🛡️ 支撑阻力:
  近期支撑: [支撑位]=[支撑值] | 近期阻力: [阻力位]=[阻力值]

🎯 综合信号:
  买卖建议: [买入/卖出/观望符号]
  风险等级: [高/中/低符号]

=================================================================
💡 提示: [总结性建议]
=================================================================
```

## AI分析增强模板格式

当启用AI分析时，在标准模板的基础上添加AI综合分析部分：

```
=================================================================
              [股票名称]([股票代码]) 技术分析报告
=================================================================
[标准报告内容 - 同上]
=================================================================

🤖 AI综合分析:
[AI生成的技术面分析]
[AI生成的短期趋势预测]
[AI生成的操作建议]
[AI生成的风险提示]

💡 提示: [总结性建议]
=================================================================
```

## 说明

- **BBI指标**: 替代了原有的RSI指标
- **位置标识**: BBI显示当前价格相对于BBI的位置（上方/下方）
- **信号标识**: 显示多头/空头状态
- **所有其他指标保持不变**: KDJ、MACD、移动平均线、知行指标等
- **AI分析**: 在标准报告基础上增加AI综合分析部分
- **API依赖**: AI分析需要配置有效的API密钥（如DeepSeek）
