# 技术分析报告模板

## 标准报告格式

运行 `python run_analysis.py 600519` 生成的报告格式：

```
=================================================================
              [股票名称]([股票代码]) 技术分析报告
=================================================================
📈 基本信息: [股票名称] | [股票代码]
💰 当前价格: [价格]元 | 涨跌: [涨跌额] | 涨幅: [涨跌幅]%

📊 技术指标概览:
  KDJ: K=[K值], D=[D值], J=[J值] | 信号: [🟢金叉/🔴死叉]
  MACD: [MACD], [信号线], [柱状图] | 信号: [🟢多头/🔴空头]
  BBI: [BBI值] | 位置: [上方/下方]
  MA5/10/20/60: [MA5]/[MA10]/[MA20]/[MA60]
  知行指标: 趋势线=[值] | 位置: [上方/下方]

🛡️ 支撑阻力:
  近期支撑: [支撑位]=[值] | 近期阻力: [阻力位]=[值]

🎯 综合信号:
  买卖建议: [🟢买入/🔴卖出/🟡观望]
  风险等级: [🔴高/🟡中/🟢低]

=================================================================
💡 提示: [操作建议]
=================================================================
```

## AI 增强报告

使用 `python run_analysis.py 600519 --ai` 时，在标准报告后追加：

```
🤖 AI综合分析:
[AI 生成的技术面分析]
[短期趋势预测]
[操作建议]
[风险提示]
```

## 指标说明

| 指标 | 说明 | 买入信号 | 卖出信号 |
|------|------|----------|----------|
| KDJ | 随机指标 | K 上穿 D (金叉) | K 下穿 D (死叉) |
| MACD | 异同移动平均线 | DIF 上穿 DEA | DIF 下穿 DEA |
| BBI | 多空指标 | 价格在 BBI 上方 | 价格在 BBI 下方 |
| 知行趋势线 | 短期趋势 | 价格突破趋势线 | 价格跌破趋势线 |

## 支撑阻力判断

| 涨跌幅 | 支撑位 | 阻力位 |
|--------|--------|--------|
| 涨幅 > 5% | MA5 | 当日最高价 |
| 跌幅 > 5% | MA20 | 开盘价 |
| -2% ~ 2% | MA20 | MA10 |
| 其他 | MA10 | MA5 |

## 风险等级

| 涨跌幅 | 风险等级 |
|--------|----------|
| > 5% | 🔴 高 |
| 2% ~ 5% | 🟡 中 |
| < 2% | 🟢 低 |
